수업 중에 사용하는 프로그램

SBS아카데미컴퓨터아트학원 광주 지점의 웹 퍼블리싱 과정에서는 수업 중에 아래에 나온 프로그램들을 사용합니다.

Visual Studio Code

HTML 문서를 작성하기 위해서는 편집기가 필요합니다. 수업 시간에는 Visual Studio Code를 사용합니다만, 다른 편집기를 사용해도 상관없습니다.

  1. 웹 브라우저로 Visual Studio Code 웹사이트에 접속해서, 화면 왼쪽에 보이는 "Download" 버튼을 눌러 설치 파일을 다운로드합니다.
    • 사용하고 있는 컴퓨터의 운영 체제에 따라 설치 파일이 자동으로 결정되지만, 다른 운영 체제의 설치 파일을 다운로드 해야 한다면 다운로드 페이지에서 원하는 운영 체제의 설치 파일을 다운로드할 수 있습니다.
  2. 다운로드한 설치 파일을 실행해서 Visual Studio Code를 설치합니다.
    • Visual Studio Code를 설치하면서 따로 설정할 것은 없습니다.
  3. 설치가 끝나면 바탕 화면에 Visual Studio Code의 바로 가기가 만들어집니다. 이 바로 가기를 더블 클릭해서 Visual Studio Code를 실행합니다.

Google Chrome

HTML 문서를 작성하면 웹 브라우저로 확인해봐야 합니다. 수업 시간에는 Google Chrome을 사용합니다만, 다른 브라우저를 사용해도 상관없습니다.

  1. 웹 브라우저로 Google Chrome 페이지에 접속해서, 설치 파일을 다운로드 합니다.
  2. 다운로드 한 설치 파일을 실행해 Google Chrome을 설치합니다.
    • Google Chrome을 설치하면서 따로 설정할 것은 없습니다.
  3. 설치가 끝나면 바탕 화면에 Google Chrome 바로 가기가 만들어집니다. 이 바로 가기를 더블 클릭해서 Google Chrome을 실행합니다.
