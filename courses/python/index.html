Visual Studio Code

파이썬 프로그램을 작성하기 위해 편집기가 필요합니다. 수업 시간에는 Visual Studio Code를 사용하지만, 다른 편집기를 사용해도 상관없습니다.

작업 디렉터리 생성

작업 디렉터리(Workspace)는 우리가 작성한 파이썬 파일을 저장하는 디렉터리(directory)를 말합니다. 작업 영역 또는 작업 폴더라고 하기도 합니다.

Microsoft Windows를 사용하는 경우, 파일 탐색기(File Explorer)를 열어 D 드라이브나 C 드라이브에 workspace 폴더를 만듭니다. 그리고 그 안에 python 폴더를 만듭니다. 그러니까 작업 디렉터리의 경로는 D:\workspace\python 또는 C:\workspace\python이 됩니다.

작업 디렉터리는 시스템 드라이브(C 드라이브)가 아닌, 다른 드라이브에 만드는 것이 좋습니다. 컴퓨터에 C 드라이브만 있다면 어쩔 수 없겠지만, C 드라이브 외에 다른 드라이브가 있다면 그 드라이브에 작업 디렉터리를 만듭니다. 비대면으로 수업을 듣는 수강생은 USB 메모리나 외장 HDD 등을 가져와서, 거기에 작업 디렉터리를 만들도록 합니다.

Tip

폴더나 파일의 이름은 한글이나 빈칸을 쓰지 않고, 영문자와 숫자, 하이픈(-), 밑줄 문자(_)로만 작성하는 게 좋습니다.

사용하는 컴퓨터의 운영 체제가 Microsoft Windows라면, 기본적으로 파일 확장자가 보이지 않도록 설정되어 있으니 파일 확장자가 보이도록 설정하는 게 좋습니다.

아래 그림과 같이 파일 탐색기의 "보기" 메뉴에서 "파일 확장명" 항목을 체크하면 파일 확장자가 표시됩니다.

Tip

파일 확장자(Filename extension)는 파일의 이름에서 파일의 종류와 그 역할을 표시하기 위해 사용하는 부분입니다. 간단하게 확장자라고도 합니다. 대부분의 운영 체제에서는 파일 이름에서 마지막 점(.) 뒤에 나오는 부분을 확장자로 인식합니다.

작업 디렉터리에서 Visual Studio Code 실행

방금 만든 작업 디렉터리를 Visual Studio Code에서 열 수 있습니다. Visual Studio Code에서는 파이썬 파일을 따로 작성하는 것보다는 이렇게 작업 디렉터리에서 작업하는 것이 더 편합니다.

Visual Studio Code에서 작업 디렉터리를 여는 방법은 여러 가지가 있습니다. 그 중에서 아래 두 가지 방법을 주로 사용합니다.

  • 파일 탐색기에서 콘텍스트 메뉴(context menu)로 작업 디렉터리를 Visual Studio Code에서 열기
  • Visual Studio Code에서 "File" 메뉴의 "Open Folder" 메뉴 항목으로 작업 디렉터리 열기

위에서 작업 디렉터리를 만들면서 실행했던 파일 탐색기가 아직 실행되어 있다면, 파일 탐색기에서 작업 디렉터리로 들어가, 마우스 오른쪽 버튼을 클릭해서 콘텍스트 메뉴를 엽니다. 콘텍스트 메뉴의 내용은 컴퓨터에 따라 다를 수 있습니다. Visual Studio Code를 설치하면서 "'Code(으)로 열기' 작업을 Windows 탐색기 디렉터리의 상황에 맞는 메뉴에 추가" 항목을 체크했다면, 아래 그림과 같이 "Code(으)로 열기"라는 메뉴 항목이 나옵니다.

Visual Studio Code에서 "File" 메뉴의 "Open Folder" 메뉴 항목으로 작업 디렉터리를 선택해서 작업 디렉터리를 열 수도 있습니다.

  3. 그 아래에 새로운 파일 항목이 추가되면, 파일 이름을 hello.py이라고 입력합니다.

Tip

앞에서 말한 것처럼, 폴더나 파일의 이름에는 한글이나 빈칸을 쓰지 않고, 영문자와 숫자, 하이픈(-), 밑줄 문자(_)로만 작성하는 게 좋습니다.

새로운 파일을 만들면, 아래 그림처럼 Visual Studio Code 화면 오른쪽, 편집기 영역에 새로 만든 파일이 열립니다.

Visual Studio Code에서 "Python" 확장 기능 설치

위 그림처럼 메시지 창이 나오지 않으면 아래 그림처럼 "Python" 확장 기능을 설치합니다.

  1. Visual Studio Code 화면 왼쪽에 있는 작업 막대(Activity Bar)에서 확장 기능(Extensions) 버튼을 클릭합니다.
  2. Visual Studio Code 화면 왼쪽 사이드 바(Side Bar) 영역에 확장 기능 관리 창이 열리면, 상단 검색 창에서 "python"으로 검색합니다.
  3. 검색 창 아래에 "Python" 확장 기능이 표시되면 클릭합니다.
  4. 편집기 영역에 "Python" 확장 기능의 설명이 표시되면, "Install" 버튼을 클릭해 확장 기능을 설치합니다.

"Python" 확장 기능을 설치하고 나면, 확장 기능 창은 닫습니다.

Tip

Visual Studio Code에서는 확장 기능(extensions)을 통해 Visual Studio Code의 기본 기능 외에도 많은 기능을 사용할 수 있으니 한번쯤 둘러보시기 바랍니다.

파이썬 프로그램 작성

새로 만든 파이썬 파일에 아래 내용을 입력합니다. 이때, 아래와 같은 사항을 주의해야 합니다.

  1. 대문자와 소문자를 구분해서 입력해야 합니다. 컴퓨터 프로그래밍에서는 대문자와 소문자를 다른 문자로 인식합니다.
  2. 오타가 나지 않도록 주의합니다.
  3. 파이썬 프로그램에서 사용하는 기호들은 정해진 의미가 있으므로 반드시 규칙대로 사용해야 합니다.
print('Hello World!')

파일을 수정하면 단축키 Ctrl+S를 눌러 저장합니다.

  1. 파일을 수정하면, Visual Studio Code 화면 왼쪽에 있는 작업 막대(Activity Bar)의 탐색기(Explorer) 버튼에 저장하지 않은 파일의 개수가 표시됩니다.
  2. 편집기 영역 상단의 파일 탭에 파일 이름이 표시됩니다. 저장하지 않은 파일은 닫기 버튼 대신 동그란 원이 표시됩니다.
  3. 편집기 영역 오른쪽 아래에는 편집기 창에 표시된 파일의 언어 모드가 표시됩니다. 이 영역을 클릭하면 편집하고 있는 파일의 언어 모드를 바꿀 수 있습니다.
  4. "Python" 확장 기능을 설치하면 Visual Studio Code에서 컴퓨터에 설치된 파이썬을 인식하고, 설치된 파이썬의 버전을 편집기 영역 오른쪽 아래에 표시합니다. 컴퓨터에 파이썬의 여러 버전이 설치되어 있으면, 이 영역을 클릭해서 원하는 버전으로 파이썬 프로그램이 실행되도록 할 수 있습니다.

Tip

Visual Studio Code에서는 다양한 파일을 편집할 수 있고, 파일 종류(Language Mode; 언어 모드)에 따라 지원하는 기능도 달라집니다. 그래서 현재 편집하고 있는 파일의 언어 모드를 제대로 지정해야 합니다.

Visual Studio Code에서는 파일 확장자를 확인해 언어 모드를 결정합니다. 파이썬 파일의 파일 확장자는 .py이니, 파일 확장자를 제대로 지정했다면 Visual Studio Code는 현재 편집기 창에 열린 파일이 파이썬 파일이라고 인식합니다.

파이썬 프로그램 실행

"Python" 확장 기능을 설치하면, 파이썬 프로그램을 작성하고 Visual Studio Code에서 바로 실행할 수 있습니다.

  1. Visual Studio Code의 편집기 영역에서 마우스 오른쪽 버튼을 클릭해 콘텍스트 메뉴를 엽니다.
  2. 콘텍스트 메뉴의 아래쪽에 있는 "Run Python" 메뉴 항목을 선택합니다.
  3. 새로 표시된 메뉴 항목들 중에서 "Run Python File in Terminal" 메뉴 항목을 선택합니다.

그러면 아래 그림과 같이 편집기 영역 아래에 내장 터미널(terminal)이 열리고 파이썬 프로그램이 실행됩니다.

아래 그림처럼 편집기 영역 오른쪽 상단에 있는 삼각형 버튼 옆에 있는 아래쪽 화살표를 눌러, 실행 메뉴가 나오면 "Run Python File" 메뉴 항목을 선택해 파이썬 프로그램을 실행할 수도 있습니다.

가장 간단한 방법은 아래 그림처럼 편집기 영역 오른쪽 상단에 있는 삼각형 버튼을 누르는 겁니다.

이와 같은 식으로 Visual Studio Code에서 파이썬 프로그램을 작성하고 실행할 수 있습니다.

그런데 수업 시간에는 프로그램을 여러 개 작성하게 되는데, 그때마다 파이썬 파일을 만드는 것은 번거러우니 주피터 노트북을 사용하기로 합니다.

Visual Studio Code에서 "Jupyter" 확장 기능 설치

주피터 노트북은 웹 브라우저에서 Julia, Python, R 등의 프로그래밍 언어로 프로그램을 작성하고 실행할 수 있는 개발 도구입니다. 프로그램 코드, 실행 결과, 마크다운(Markdown)으로 작성한 문서, 이미지 등을 결합해서 "작업을 보여주는 데" 유용하며, 셀별로 프로그램 코드를 실행해 코드의 기능을 이해할 수 있도록 도와줍니다.

아나콘다(Anaconda)를 설치하면 주피터 노트북이 함께 설치되며, pip를 이용해 Jupyter 패키지를 설치할 수도 있습니다. 하지만 아나콘다를 설치하지 않아도, Visual Studio Code에서 "Jupyter" 확장 기능을 설치하면 주피터 노트북을 사용할 수 있습니다.

  1. Visual Studio Code 화면 왼쪽에 있는 작업 막대(Activity Bar)에서 확장 기능(Extensions) 버튼을 클릭합니다.
  2. Visual Studio Code 화면 왼쪽 사이드 바(Side Bar) 영역에 확장 기능 관리 창이 열리면, 상단 검색 창에서 "jupyter"로 검색합니다.
  3. 검색 창 아래에 "Jupyter" 확장 기능이 표시되면 클릭합니다.
  4. 편집기 영역에 "Jupyter" 확장 기능의 설명이 표시되면, "Install" 버튼을 클릭해 확장 기능을 설치합니다.

"Jupyter" 확장 기능을 설치하고 나면, 확장 기능 창은 닫습니다.

새로운 주피터 노트북 생성

"Jupyter" 확장 기능을 설치하고 나서 새로운 주피터 노트북을 만듭니다.

  1. Ctrl+Shift+P 단축키를 눌러 명령어 팔레트(Command Palette) 창을 엽니다.
  2. Visual Studio Code 화면 상단에 명령어 팔레트가 표시되는데, 검색 창에 "jupyter"를 입력해 검색합니다.
  3. "Jupyter"가 포함된 명령어들이 나오면, 그 중에서 "Create: New Jupyter Notebook" 메뉴 항목을 선택해 새로운 주피터 노트북을 만듭니다.

Tip

명령어 팔레트(Command Palette)는 Visual Studio Code에서 제공하는 모든 명령어들을 담고 있습니다. 필요한 기능이 기억나지 않으면 명령어 팔레트에서 찾을 수 있습니다.

그러면 아래와 같이 편집기 영역에 새로운 주피터 노트북이 만들어집니다.

이렇게 명령어 팔레트에서 명령어를 찾아 주피터 노트북을 만들면 파일 이름을 지정해서 저장해야 합니다. 이렇게 하는 건 번거러우니 파이썬 파일을 만드는 것처럼 탐색기 창에서 주피터 노트북을 추가합니다. 이때 주의할 것은 파일 확장자입니다. 주피터 노트북의 파일 확장자는 .ipynb입니다.

  1. Visual Studio Code 화면 왼쪽에 있는 작업 막대(Activity Bar)에서 탐색기(Explorer) 버튼을 클릭합니다.
  2. Visual Studio Code 화면 왼쪽 사이드 바(Side Bar) 영역에 탐색기 창이 열리면, 아래쪽 작업 디렉터리(workspace)의 파일이 표시되는 영역에 마우스 포인터를 가져가서 위쪽에 표시되는 "New File" 버튼을 클릭합니다.
  3. 그 아래에 새로운 파일 항목이 추가되면, 파일 이름을 day01.ipynb라고 입력합니다.

그러면 아래 그림과 같이 새로운 주피터 노트북이 만들어집니다.

그림에서 보이는 1번 영역을 코드 셀(Cell)이라고 부릅니다. 이 코드 셀에 파이썬 프로그램을 작성하고 실행할 수 있습니다. 주피터 노트북에서는 이러한 코드 셀을 여러 개 넣을 수 있습니다. 그래서 파이썬 프로그래밍을 공부할 때 정말 편리합니다.

주피터 노트북에서 파이썬 프로그램을 실행하려면 먼저 커널(kernel)을 설정해야 합니다.

주피터 노트북 화면 상단에 있는 "Select Kernel" 버튼을 누릅니다. 그러면 아래 그림과 같이 Visual Studio Code 화면 상단에 커널을 선택할 수 있는 창이 나옵니다.

커널 선택 창에서 "Python Environment" 항목을 선택합니다.

그러면 위 그림처럼, 앞에서 설치했던 파이썬이 화면에 표시될 텐데 그걸 선택합니다.

주피터 노트북의 커널을 설정하면 위 그림처럼 주피터 노트북 화면 상단에 있던 "Select Kernel" 버튼이 바뀝니다. 이제 주피터 노트북의 코드 셀에 파이썬 프로그램을 작성하고 실행할 수 있습니다.

주피터 노트북의 코드 셀에 파이썬 프로그램을 작성하고 실행

주피터 노트북의 코드 셀에 아까 작성했던 프로그램을 작성합니다.

주피터 노트북의 코드 셀에 파이썬 프로그램을 작성하고, 코드 셀의 왼쪽 상단에 있는 실행 버튼을 누르면 프로그램이 실행됩니다. 프로그램 실행이 끝나면 코드 셀 아래에 실행 결과가 출력됩니다.

이처럼 주피터 노트북을 이용하면 단위 프로그램을 쉽게 작성하고 실행할 수 있습니다.

  • 주피터 노트북은 셀(Cell)로 구성되며, 셀에는 코드 셀(code cell)과 마크다운 셀(markdown cell)이 있습니다.
  • 코드 셀(Code cell)에는 프로그램 코드를 작성하며, 개별적인 셀을 실행해서 결과를 확인할 수 있습니다.
    • 코드 셀의 마지막 줄에 표현식이나 값이 나오면, 대화형 쉘(interactive shell)이나 IDLE에서처럼 그 값을 출력합니다.
  • 마크다운 셀(Markdown cell)에는 마크다운(Markdown) 문법으로 간단한 글을 작성합니다.
