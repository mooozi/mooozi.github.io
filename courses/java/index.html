수업 중에 사용하는 프로그램

수업 중에는 아래에 나온 프로그램들을 사용합니다.

  • Adoptium Eclipse Temurin JDK
  • Visual Studio Code

Adoptium Eclipse Temurin JDK

자바 프로그램을 작성하고 실행하기 위해서는 JDK(Java Development Kit)가 필요합니다. 수업 시간에는 OpenJDK 중 하나인 Adoptium Eclipse Temurin JDK를 사용합니다.

Temurin JDK 설치 프로그램 다운로드

웹 브라우저로 Adoptium 웹사이트에 접속해서, 화면 왼쪽에 보이는 "Latest LTS Release" 버튼을 클릭해 사용하고 있는 운영 체제에 맞게 Temurin JDK 최신 버전을 다운로드합니다.

Temurin JDK 설치

다운로드한 설치 프로그램을 실행해서 Adoptium Eclipse Temurin JDK를 설치합니다.

"Custom Setup" 화면에서 JAVA_HOME 환경 변수를 추가하기 위해 아래와 같이 설정합니다.

  1. "Set or override JAVA_HOME variable" 항목 왼쪽에 있는 버튼을 클릭합니다.
  2. 메뉴가 표시되면 "Will be installed on local hard drive" 항목을 선택합니다.
  3. "Next" 버튼을 클릭해 설치를 계속 진행합니다.

Temurin JDK 설치 후 확인

Adoptium Eclipse Temurin JDK가 제대로 설치됐는지 확인하기 위해 Win+R 키를 눌러 실행 창을 엽니다.

위 그림과 같이 실행 창에서 cmd를 입력해 명령 프롬프트를 실행합니다.

명령 프롬프트가 실행되면 javac --version 명령을 입력하고 엔터 키를 누릅니다. 설치한 Temurin JDK의 버전 정보가 출력되면 제대로 설치된 겁니다.

Visual Studio Code

자바 프로그램을 작성하기 위해 편집기가 필요합니다. 수업 시간에는 Visual Studio Code를 사용하지만, 다른 편집기를 사용해도 상관없습니다.

Visual Studio Code 다운로드

웹 브라우저로 Visual Studio Code 웹사이트에 접속해서, 화면 왼쪽에 보이는 "Download" 버튼을 눌러 설치 파일을 다운로드합니다.

컴퓨터의 운영 체제에 따라 설치 파일이 다릅니다. Visual Studio Code 웹사이트에 접속하면 사용하는 컴퓨터의 운영 체제에 따라 설치 파일이 자동으로 결정됩니다. 다른 운영 체제의 설치 파일이 필요하면, 다운로드 페이지에서 원하는 운영 체제의 설치 파일을 다운로드할 수 있습니다.

Visual Studio Code 설치

다운로드한 설치 파일을 실행해서 Visual Studio Code를 설치합니다. Visual Studio Code를 설치하면서 따로 설정할 것은 없습니다. 다만, 아래 그림과 같이 "추가 작업 선택" 화면에서 모든 항목을 체크하고 설치하는 게 좋습니다.

Visual Studio Code 실행

위 그림과 같이 체크하고 설치했다면, 설치가 끝나고 바탕 화면에 Visual Studio Code의 바로 가기가 만들어집니다. 이 바로 가기를 더블 클릭해서 Visual Studio Code를 실행합니다.

개발 환경 설정

이제 Visual Studio Code로 자바 프로그램을 작성하고 실행하기 위해 개발 환경을 설정합니다. 개발 환경을 설정하는 순서는 아래와 같습니다.

  1. 작업 디렉터리 생성
  2. 작업 디렉터리에서 Visual Studio Code 실행
  3. Visual Studio Code에서 "Java" 확장 기능 설치
  4. 자바 프로젝트 생성
  5. 자바 프로그램 작성
  6. 자바 프로그램 실행

작업 디렉터리 생성

작업 디렉터리(Workspace)는 우리가 작성한 파이썬 파일을 저장하는 디렉터리(directory)를 말합니다. 작업 영역 또는 작업 폴더라고 하기도 합니다.

자바 프로젝트는 작업 디렉터리 안에 만듭니다. 자바 프로젝트에는 패키지들이 들어가고, 패키지에 자바 클래스를 만듭니다.

Microsoft Windows를 사용하는 경우, 파일 탐색기(File Explorer)를 열어 D 드라이브나 C 드라이브에 workspace 폴더를 만듭니다. 그리고 그 안에 java 폴더를 만듭니다. 그러니까 작업 디렉터리의 경로는 D:\workspace\java 또는 C:\workspace\java가 됩니다.

작업 디렉터리는 시스템 드라이브(C 드라이브)가 아닌, 다른 드라이브에 만드는 것이 좋습니다. 컴퓨터에 C 드라이브만 있다면 어쩔 수 없겠지만, C 드라이브 외에 다른 드라이브가 있다면 그 드라이브에 작업 디렉터리를 만듭니다. 비대면으로 수업을 듣는 수강생은 USB 메모리나 외장 HDD 등을 가져와서, 거기에 작업 디렉터리를 만들도록 합니다.

Tip

폴더나 파일의 이름은 한글이나 빈칸을 쓰지 않고, 영문자와 숫자, 하이픈(-), 밑줄 문자(_)로만 작성하는 게 좋습니다.

사용하는 컴퓨터의 운영 체제가 Microsoft Windows라면, 기본적으로 파일 확장자가 보이지 않도록 설정되어 있으니 파일 확장자가 보이도록 설정하는 게 좋습니다.

아래 그림과 같이 파일 탐색기의 "보기" 메뉴에서 "파일 확장명" 항목을 체크하면 파일 확장자가 표시됩니다.

Tip

파일 확장자(Filename extension)는 파일의 이름에서 파일의 종류와 그 역할을 표시하기 위해 사용하는 부분입니다. 간단하게 확장자라고도 합니다. 대부분의 운영 체제에서는 파일 이름에서 마지막 점(.) 뒤에 나오는 부분을 확장자로 인식합니다.

자바 파일의 파일 확장자는 .java입니다.

작업 디렉터리에서 Visual Studio Code 실행

방금 만든 작업 디렉터리를 Visual Studio Code에서 열 수 있습니다. Visual Studio Code에서 작업 디렉터리를 여는 방법은 여러 가지가 있습니다. 그 중에서 아래 두 가지 방법을 주로 사용합니다.

  • 파일 탐색기에서 콘텍스트 메뉴(context menu)로 작업 디렉터리를 Visual Studio Code에서 열기
  • Visual Studio Code에서 "File" 메뉴의 "Open Folder" 메뉴 항목으로 작업 디렉터리 열기

위에서 작업 디렉터리를 만들면서 실행했던 파일 탐색기가 아직 실행되어 있다면, 파일 탐색기에서 작업 디렉터리로 들어가, 마우스 오른쪽 버튼을 클릭해서 콘텍스트 메뉴를 엽니다. 콘텍스트 메뉴의 내용은 컴퓨터에 따라 다를 수 있습니다. Visual Studio Code를 설치하면서 "'Code(으)로 열기' 작업을 Windows 탐색기 디렉터리의 상황에 맞는 메뉴에 추가" 항목을 체크했다면, 아래 그림과 같이 "Code(으)로 열기"라는 메뉴 항목이 나옵니다.

Visual Studio Code에서 "File" 메뉴의 "Open Folder" 메뉴 항목으로 작업 디렉터리를 선택해서 작업 디렉터리를 열 수도 있습니다.

Visual Studio Code에서 "Java" 확장 기능 설치

Visual Studio Code에서 자바 프로그램을 작성하고 실행하기 위해 "Java" 확장 기능을 설치합니다.

  1. Visual Studio Code 화면 왼쪽에 있는 작업 막대(Activity Bar)에서 확장 기능(Extensions) 버튼을 클릭합니다.
  2. Visual Studio Code 화면 왼쪽 사이드 바(Side Bar) 영역에 확장 기능 관리 창이 열리면, 상단 검색 창에서 "java"로 검색합니다.
  3. 검색 창 아래에 "Java" 확장 기능이 표시되면 클릭합니다.
  4. 편집기 영역에 "Java" 확장 기능의 설명이 표시되면, "Install" 버튼을 클릭해 확장 기능을 설치합니다.

"Java" 확장 기능을 설치하고 나면, 확장 기능 창은 닫습니다.

새 자바 프로젝트 생성

"Java" 확장 기능을 설치하고, 작업 디렉터리에 새 자바 프로젝트를 만듭니다. 자바 프로젝트는 자바 애플리케이션을 만들기 위한 단위입니다. 하지만 수업 시간에는 과정이 진행되는 동안 이 프로젝트 안에 자바 클래스들(Java classes)을 작성하게 됩니다.

  1. Visual Studio Code 화면 왼쪽에 있는 작업 막대(Activity Bar)에서 탐색기(Explorer) 버튼을 클릭합니다.
  2. Visual Studio Code 화면 왼쪽에 보이는 작업 영역(Workspace)에서 마우스 오른쪽 버튼을 클릭해서 콘텍스트 메뉴(context menu)를 엽니다.
  3. 콘텍스트 메뉴가 열리면 "New Java Project" 메뉴 항목을 선택합니다.

그러면 위 그림과 같이 프로젝트 형식을 선택하기 위한 창이 표시됩니다. 수업 시간에는 빌드 도구를 사용하지 않으니 "No build tools" 항목을 선택합니다.

프로젝트 형식을 결정하면, 프로젝트를 저장할 폴더를 지정하는 창이 나옵니다. 조금 전에 만들었던 작업 디렉터리를 선택합니다. 그러면 프로젝트는 작업 디렉터리 안에 만들어집니다.

다음으로는 프로젝트 이름을 지정합니다. 프로젝트 이름은 파스칼 표기법(Pascal notation; 각 단어의 첫 글자를 대문자로 표기)으로 작성합니다. 여기에서는 JavaStudy로 지정했습니다.

프로젝트 이름까지 지정하면, 프로젝트가 만들어지고 새 창으로 프로젝트가 열립니다. 새로 만든 프로젝트에는 README.md 파일과 .vscode, lib, src 폴더가 있습니다.

  • README.md 파일 ― 프로젝트 소개 등을 작성하는 파일
  • .vscode 폴더 ― Visual Studio Code의 설정 파일(settings.json)이 저장된 폴더
  • lib 폴더 ― 프로젝트에서 필요한 외부 라이브러리를 저장하는 폴더
  • src 폴더 ― 우리가 작성하는 자바 클래스들을 저장하는 폴더

자바 프로그램은 자바 클래스들로 이루어져 있습니다. 일반적으로 자바 클래스들은 패키지에 포함되며, 패키지는 src 폴더에 만듭니다.

자바 프로그램 작성

자바 프로그램을 작성한다는 것은 자바 클래스를 만든다는 말과 같습니다. 앞에서 말했듯 자바 클래스는 패키지(package)에 포함됩니다. 그래서 자바 클래스를 만들려면 먼저 패키지부터 만들어야 합니다. 패키지는 폴더로 만들어지며, src 폴더에 만듭니다.

  1. 탐색기 창에서 src 폴더를 선택한 다음, 마우스 오른쪽 버튼을 눌러 콘텍스트 메뉴를 엽니다.
  2. 콘텍스트 메뉴에서 "New Java Package..." 항목을 눌러 day01 패키지를 추가합니다.

패키지를 만들고 나서, 패키지에 자바 클래스를 만듭니다.

  1. 탐색기 창에서 방금 만든 day01 패키지 선택합니다.
  2. 마우스 오른쪽 버튼을 눌러 콘텍스트 메뉴를 표시합니다.
  3. "New Java File" 메뉴 항목을 선택하고, "Class..." 메뉴 항목을 선택합니다.

새로운 자바 클래스를 추가하면 아래 그림과 같이 클래스 이름을 지정하는 창이 나옵니다.

자바 클래스의 이름을 HelloTest로 입력합니다. 그러면 아래 그림처럼 HelloTest.java 파일이 day01 패키지에 만들어집니다.

새로운 자바 클래스를 추가하면 위 그림처럼 package 선언과 클래스 정의가 자동으로 추가됩니다.

이 파일에 아래 내용을 입력합니다. 이때, 아래 사항들을 주의해야 합니다.

  1. 대문자와 소문자를 구분해서 입력해야 합니다. 컴퓨터 프로그래밍에서는 대문자와 소문자를 다른 문자로 인식합니다.
  2. 오타가 나지 않도록 주의합니다.
  3. 자바 프로그램에서 사용하는 기호들은 정해진 의미가 있으므로 반드시 규칙대로 사용해야 합니다.
package day01;

public class HelloTest {
    public static void main(String[] args) {
        System.out.println("Hello World!");
    }
}

파일을 수정하면 단축키 Ctrl+S를 눌러 저장합니다.

편집기 영역 오른쪽 아래에는 편집기 창에 표시된 파일의 언어 모드가 표시됩니다. 이 영역을 클릭하면 편집하고 있는 파일의 언어 모드를 바꿀 수 있습니다.

Tip

Visual Studio Code에서는 다양한 파일을 편집할 수 있고, 파일 종류(Language Mode; 언어 모드)에 따라 지원하는 기능도 달라집니다. 그래서 현재 편집하고 있는 파일의 언어 모드를 제대로 지정해야 합니다.

Visual Studio Code에서는 파일 확장자를 확인해 언어 모드를 결정합니다. 자바 파일의 파일 확장자는 .java이니, 새로운 자바 클래스를 추가하면 Visual Studio Code는 현재 편집기 창에 열린 파일이 자바 파일이라고 인식합니다.

자바 프로그램 실행

"Java" 확장 기능을 설치하면, 자바 프로그램을 작성하고 Visual Studio Code에서 바로 실행할 수 있습니다.

  1. Visual Studio Code 화면 상단의 메뉴 바에서 "Run" 메뉴 항목을 선택합니다.
  2. 하위 메뉴가 표시되면 "Run Without Debugging" 메뉴 항목을 선택해 자바 프로그램을 실행합니다.
  3. 또는 Ctrl+F5 단축키를 눌러 자바 프로그램을 실행할 수 있습니다. 앞으로 많이 사용할 단축키이니 외워둡시다.

그러면 아래 그림과 같이 편집기 영역 아래에 내장 터미널(terminal)이 열리고 자바 프로그램이 실행됩니다.

아래 그림처럼 편집기 영역 오른쪽 상단에 있는 삼각형 버튼 옆에 있는 아래쪽 화살표를 눌러, 실행 메뉴가 나오면 "Run Java" 메뉴 항목을 선택해 자바 프로그램을 실행할 수도 있습니다.

또는 아래 그림처럼 편집기 영역 오른쪽 상단에 있는 삼각형 버튼을 클릭하면 자바 프로그램을 실행할 수 있습니다.

main 메서드 위에 표시된 "Run" 항목을 클릭해서 자바 프로그램을 실행할 수도 있습니다.

이처럼 여러 방법으로 Visual Studio Code에서 자바 프로그램을 실행할 수 있는데, 아무래도 가장 간단한 방법은 Ctrl+F5 단축키를 누르는 것일 겁니다.
